# 石器時代 ASSA 外掛輔助程式

**此程式是以凝夢閣為目的開發的, 有許多功能只適用這個伺服器**

這個輔助程式可以協助多開的你, 幫你整理腳色基本資訊在同一個介面  
同時也有一些方便的功能讓你的遊戲體驗更加順暢:  
寵物成長計算機 - 自動計算寵物成長率, 並提供右鍵選單方便分享/紀錄寵物資訊  
激素數值預測 - (預設為關閉請至設定開啟) 可預測 150 - 200 之間的等級數值 (顯示在寵物右鍵選單)  
補運小幫手 - 自動顯示補運道具怎麼拿, 寵物哪裡抓省去查論壇的麻煩  
補運自動料理 - 自動產生補運料理的製作腳本並傳到外掛自動執行  
石器視窗大小調整 - 調整石器視窗的大小, 僅限於掛端  
一級寵物遇敵記錄 - 幫你記下一級寵物的遇敵地圖與座標  
寵物選單 - 點選寵物的狀態框框會自動開啟外掛的寵物選單  
自動改寵名腳本 - 壓縮檔內附上自動把寵物名稱改成四圍的腳本, 方便使用成長計算機功能  
希望各位用的愉快~ 也歡迎回報錯誤或者提供補運資料庫不足的資料  

**此程式不是取代ASSA用的，而是需要與ASSA並行使用**

## 安裝
解壓縮後程式資料夾你要放哪裡都可以, 但如果你有裝防毒他可能會抱怨  
所以建議丟石器資料夾裡面某處 (因為這裡應該已經被你設例外了)  

## 使用說明
先開啟全腳色的Assa外掛, 再開啟這個輔助程式SaBotSupport.exe  
點擊左上角Attach按鈕, 就會自動擷取外掛資訊  
![](https://i.imgur.com/q2pyqc5.gif)
如果你外掛關了重開, 可以重新按Attach就好  
如果你Assa需要管理員權限才能開, 那麼這個輔助程式也必須要用管理員權限開喔不然會抓不到  
(補充一下, Assa的相容性可以設Windows 7這樣就不用管理員權限了, 試過Windows 10下可以跑)  
(補充: 如果相容性設Windows 7不能跑 可以試試看先設XP3跑一次可以開之後 再改回Windows 7)  
![](https://i.imgur.com/SKrL5w6.jpg)

## 寵物成長計算機
寵物名稱如果改成一級時的四圍, 那本程式就會自動幫你計算成長率(即時更新)  
如果沒有改暱稱, 則會改使用內建的寵物成長紀錄做計算(前提是寵物升級時程式是開著的)  
在寵物欄開啟右鍵選單可以查看基礎數值以及預測等級的面板數值, 且會自動複製寵物數值方便貼到別的地方(可自訂格式)  
![](https://i.imgur.com/q4KltSY.gif)
那如果像是野生寵物沒有一級素質怎麼辦? 沒關係可以暱稱改成野生素質後面加底線跟等級, 就可以計算了  
舉例:  
14.9.8.7 (一級寵物)  
14.9.8.7威威 (一級寵物, 自動忽略簡稱)  
249.76.55.45_35 (35級寵物)  
要注意的是, 寵物等級超過100會因為暱稱長度不夠塞而不能改名, 所以請使用以下連結內的特殊命名規格  
那這些要手動改名嗎? No那樣太累了, 請服用我寫的自動改寵名腳本  
二轉140或以上的寵物可以預測吃激素後的數值 (預設為關閉需在設定中開啟)  
分為一般寵物模式與寵物王模式, 程式不會自動判斷是否寵物可以轉王需要手動設定  
![](https://i.imgur.com/msDYNxK.jpg)
![](https://i.imgur.com/ze4RI6Y.jpg)

## 補運小幫手
道具欄介面滑鼠移到補運道具上方, 就會自動查詢資料庫並顯示補運道具或寵物的入手方式  
在補運道具上按右鍵可以開啟道具選單. 有多個選項可以選擇  
將目標寵物的名稱複製到剪貼簿的選項是方便設定外掛的自動抓寵  
如果任務內容包含料理, 則會出現自動生成腳本的按鈕, 點一下程式就會自動寫一個該料理的腳本並傳送到Assa自動執行  
![](https://i.imgur.com/sm0mvQV.gif)
若遇到沒有資料或資料錯誤的情況, 請多多回報讓我修改喔 (Discord 找 Setsu)  
(資料庫是個簡單的csv格式, 如果我忙沒空改可以自行編輯)

程式上方選單可開啟石器視窗調整功能

## 遊戲對話擷取
同ASSA的聊天分頁, 支援發送對話以及更改對話顏色  
除此之外可以藉由開關過濾掉不想看的訊息種類 (像是公告, 系統訊息 ...等)  
其中合併顯示模式會將所有腳色的對話內容同時顯示 (來自其他腳色的訊息會向右校正)  
對話只會保留最後兩小時的內容, 其餘會備份至 /Windows帳號/AppData/Local/SABotSupport/亂碼/Chatlog  
切換腳色時為了避免介面停頓過久只會對最新的100筆訊息進行上色/校正, 其他會全部以預設的黑色顯示  

## 自動登陸功能
在上方隱藏式工具列開啟介面選單  
輸入要登入的帳號/密碼/分流/腳色, 再設定要登入的ASSA路徑位置  
如果已經有Attach, 則會自動載入到ASSA路徑表格  
如果ASSA尚未運行(但是路徑有設定)會自行開啟  
可記錄帳號密碼 (預設為關閉請自行勾選啟用) (按右上角的 X 則不會記錄直接關閉)  
可按設定分流選擇serverlist.ini檔案來載入不同分流名稱 (如果未來該檔案有更新, 必須重新設定)  

## 人物狀態紀錄
會自動紀錄聲望, 活力, VIP狀態, 上線時間, 經驗上升時間...等數值  
但是需要手動使用/status指令才會更新

## Q&A

### 1. 一直顯示抓不到ASSA怎麼辦?  
試試看用管理員執行, 還是不行可以去設定勾選使用舊版擷取模式 (但是這個模式比較慢一點)  
如果你將ASSA隱藏了也會抓不到  

### 2. 壓縮檔案無法解壓縮?  
請使用新一點的解壓縮程式, 比方說7zip  

### 3. 請問支援32位元作業系統嗎? 請問支援XP嗎?  
支援32位元作業系統  
不支援XP (強烈建議不要再用XP了)  
最低作業系統支援是Windows 7 SP1  

### 4. 為什麼補運按右鍵只有顯示複製寵物名稱, 然後提示框都是"沒有資料"?  
你可能是從比較舊的版本用覆蓋的方式更新成新的版本  
把資料夾內跟主程式(.exe)在同一個資料夾層級的.dll檔案都刪除即可 (lib子資料夾內的不可以刪除!)
